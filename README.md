### Compile

~/sor/build$ cmake .. <br>
~/sor/build$ make

### 실행

~/sor/build$ ./lec07_sor


SOR로 노이즈 제거 후 <br>
z축 제거해서 2D 이미지로 저장. <br>
Make 파일 만들어서 sor 하나만 CMake로 돌아가도록 만듦.

Input: pcd <br>
Output: point cloud visualization, 2d_projection.png
